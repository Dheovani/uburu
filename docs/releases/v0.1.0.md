# Uburu v0.1.0 — Windows Preview

This is the first public Windows preview of Uburu, a local desktop search tool focused on fast file search, responsive previews, and support for developer-oriented folders and repositories.

This release includes a Windows installer and an optional portable ZIP package. It supports direct search, indexed search foundations, Git-aware architecture, document-content extraction for common formats such as PDF, DOCX, XLSX, PPTX, ODT, RTF, HTML, and plain text, plus a Qt/QML desktop interface with preview, highlighting, history, favorites, and cancellation.

This is still a preview release: binaries are not code-signed yet, Linux/macOS packages are not published yet, and performance tuning for very large repositories is still ongoing.

## Recommended download

- `uburu-setup-<version>-windows-x64.exe`: Windows installer.
- `uburu-setup-<version>-windows-x64.exe.sha256`: checksum for the installer.

## Optional portable download

- `uburu-windows-msvc-x64.zip`: portable Windows bundle.
- `uburu-windows-msvc-x64.sha256`: checksum for the portable bundle.

## Validation performed

- Windows Release bundle generated with Qt, MSVC runtime, and vcpkg DLLs.
- Windows installer generated with Inno Setup 6.
- Installer install, launch, basic search, and uninstall validated manually on Windows.

## Known limitations

- Release binaries are not code-signed yet.
- Linux and macOS packages are not published yet.
- Large-repository performance still needs broader validation and tuning.
- Update and rollback flows are not automated yet.
